# 射擊遊戲

一個簡單但刺激的射擊遊戲，考驗玩家的反應能力和準確度。

## 遊戲特點

- 簡單易上手的操作
- 精美的視覺效果
- 刺激的遊戲體驗

## 操作說明

- 使用方向鍵移動飛機
- 空白鍵發射子彈
- 擊敗敵人獲得高分

## 技術實現

- HTML5 Canvas
- JavaScript
- CSS3 動畫效果

## 開發文檔

### 主要文件結構
```
game-1/
├── index.html      # 遊戲主頁面
├── style.css       # 遊戲樣式
└── script.js       # 遊戲邏輯
```

### 核心功能
1. 玩家控制
2. 敵人生成
3. 碰撞檢測
4. 分數系統
5. 音效系統
